Complete

The data is scraped from Instagram since official API doesn't support location information of where the photo post is taken. When instagram user post each photo, they can add many tags to make the post more searchable, so I scrape the basic information (user name, location is any, post time, etc) of all the photo which has been attached with a "Letote" tag and "Renttherunway" tag. As the screenshot shown below, the user "lananamendez" has tagged "Letote" in her post at Chicago Riverwalk.

Not all user post the location of their photo, and here is an overview of data I obtained:

  1. Renttherunway: 16,597 posts with location out of 41,352 posts (last update 4/7/2018)
  2. Letote: 3,556 posts with location out of 14,080 posts (last update 4/7 2018)
